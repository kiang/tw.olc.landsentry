上架 Play Console 時各必填欄位的建議內容，可直接複製貼上後微調。

## 應用程式名稱（上限 30 字元）

```
國土巡守隊 LandSentry
```

## 簡短說明（上限 80 字元）

```
全台國土監測變異點地圖：實地巡查、拍照記錄、追蹤違規改善進度，與隊友雲端分享。
```

## 完整說明（上限 4000 字元）

```
國土巡守隊（LandSentry）讓關心土地的你，把政府公開的國土利用監測資料帶著走。

內政部透過衛星影像比對，持續找出全台土地利用的「變異點」——可能是違規開發、盜採土石、未經許可的整地或填土。這些資料是公開的，但散落在網頁裡不易追蹤。國土巡守隊把它們下載到你的手機，讓你在地圖上巡查、實地走訪、拍照記錄，持續關注每個點位是否改善。

【主要功能】

🗺 全台變異點地圖
以國土測繪中心電子地圖與正射影像（空照圖）為底圖，紅色標記代表查證「違規」案件。支援縣市、年份、變異類型、查證結果四種篩選，鄰近點位自動聚合。

📋 清單瀏覽與搜尋
依變異點編號、類型、權責單位快速搜尋，一鍵只看違規案件。

⭐ 追蹤管理
把需要關注的點位加入追蹤，以五種狀態管理：追蹤中、已改善、無變化、惡化中、已結案。每次回訪留下一筆時間軸紀錄。

📷 照片紀錄
拍照或從相簿加入現場照片，附在追蹤紀錄上，隨時全螢幕檢視比對。

📍 GPS 定位與導航
地圖顯示所在位置與各點位距離，一鍵開啟 Google Maps 導航前往現場。

🤝 團隊協作分享
將追蹤狀態、紀錄與照片匯出成單一檔案，存到 Google 雲端硬碟或傳給隊友；對方匯入後自動合併，重複匯入不會產生重複資料。

📴 離線可用
變異點資料下載後儲存於手機本機，山區巡查沒有網路也能記錄（地圖底圖需要網路）。

【資料來源】
內政部國土利用監測整合資訊網（landchg.tcd.gov.tw）公開資料，經 kiang.github.io 整理為開放格式。地圖底圖由國土測繪圖資服務雲提供。

【隱私承諾】
本 app 沒有伺服器，不收集、不上傳任何個人資料。所有追蹤紀錄、照片與定位資訊僅儲存在你的裝置；分享功能完全由你主動操作。

守護土地，從看見開始。
```
