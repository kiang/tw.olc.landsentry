# 使用指南 (Usage Guide)

LandSentry 國土巡守隊：下載國土利用監測變異點、在地圖上巡查，
並以照片與狀態紀錄持續追蹤每個點位的改善情形。

App 底部有四個分頁：**地圖**、**清單**、**追蹤**、**資料**。

---

## 第一次使用

1. 開啟 app 後預設顯示「臺南市、最新年份」。尚未下載資料時，
   地圖中央會出現下載提示卡片，點「**立即下載**」即可。
2. 想看其他縣市或年份，用畫面上方的篩選列切換；沒下載過的組合
   一樣會出現下載提示。

---

## 地圖

- **底圖**為國土測繪中心 (NLSC) 電子地圖；點右下角衛星圖示
  可切換成**正射影像**（空照圖），再點一次切回。
- **標記顏色**：紅色 = 查證結果「違規」；藍色 = 非違規或其他。
  數字圓圈是聚合標記（含違規點時顯示紅色），點它會放大展開。
- 已加入追蹤的點位右上角會有 ⭐ 星號。
- 上方篩選列可切換 **縣市 / 年份 / 變異類型**；下方圖例顯示目前點位總數。
- 右下角按鈕由上而下：切換底圖、顯示全部點位（回到全覽範圍）、
  **我的位置**（GPS 定位，地圖移到所在位置並顯示藍點）。
- **點擊標記**開啟摘要卡：變異類型、編號、查證結果、距離
  （已定位時），並可直接「詳細與紀錄」、「導航」或「加入追蹤」。

## 清單

- 與地圖共用同一組縣市／年份／類型篩選。
- 搜尋框可輸入**編號、變異類型或權責單位**關鍵字過濾。
- 「違規」籤可一鍵只看違規點。
- 點任一列進入詳細頁。

## 變異點詳細頁

- 上方為該點的**正射影像縮圖**與完整欄位資料（查證結果以顏色標示），
  右上角可複製座標。
- 外部連結：**導航前往**（顯示與目前位置的距離，開啟 Google Maps
  導航）、**Google 衛星**、**Bing 衛星**、**案件詳情**
  （landchg.olc.tw 案件頁）。

## 追蹤與紀錄

追蹤是本 app 的核心：把需要持續關注的點位收進「追蹤」分頁，
每次回訪都留下一筆紀錄。

1. **加入追蹤**：在地圖摘要卡按「加入追蹤」，或在詳細頁選擇狀態後
   按「開始追蹤」。
2. **追蹤狀態**共五種：
   | 狀態 | 意義 |
   |------|------|
   | 🟠 追蹤中 | 持續觀察中 |
   | 🟢 已改善 | 現場已改善 |
   | ⚪ 無變化 | 回訪後無明顯變化 |
   | 🔴 惡化中 | 情況持續惡化 |
   | ⚫ 已結案 | 不再需要追蹤 |
3. **新增紀錄**：在詳細頁選狀態、輸入現場觀察文字，可用
   「**拍照**」（相機）或「**相簿**」（系統相片選擇器）附加照片，
   按「新增紀錄」存檔。紀錄會以時間軸顯示，照片縮圖點開可全螢幕
   縮放檢視。
4. **追蹤分頁**：列出所有追蹤中的點位，可依狀態篩選，每張卡片顯示
   最新紀錄摘要與照片數；點卡片回到詳細頁。
5. **取消追蹤**：詳細頁右上角 ⭐ 星號 → 確認後移除追蹤
   （歷史紀錄會保留，重新追蹤即可再看到）。

> 所有紀錄與照片只存在手機本機，不會自動上傳。

## 資料分頁（下載與管理）

- **下載資料集**：選縣市與年份（民國）後按「下載」。
- **已下載資料集**：顯示每組資料的筆數與更新時間；
  - 🔄 重新下載（更新為最新資料，追蹤紀錄不受影響）
  - 🗑 刪除（追蹤中的點位與紀錄會保留）
  - 點整列可直接把地圖／清單切到該資料集。

## 追蹤資料分享（Google 雲端硬碟）

跨使用者協作：把自己的追蹤成果分享給隊友，或合併隊友的紀錄。

### 匯出

1. 資料分頁 →「追蹤資料分享」→「**匯出分享**」。
2. App 會把所有追蹤狀態、紀錄、相關變異點與照片打包成單一
   `landsentry_日期.zip`，並開啟系統分享面板。
3. 選 **雲端硬碟 (Drive)** 存到指定資料夾，或用 Gmail / 通訊軟體
   直接傳給對方。

### 匯入

1. 資料分頁 →「**匯入合併**」，開啟系統檔案選擇器。
2. 從左側選單進入 **雲端硬碟**，瀏覽到對方分享的資料夾，
   選取 zip 檔。
3. 匯入採**合併**而非覆蓋：
   - 追蹤狀態：同一點位兩邊都有時，**較新的**那筆生效。
   - 紀錄：以 UUID 去重後**聯集**，重複匯入同一檔案不會產生重複資料。
   - 照片：缺少的才複製。
   - 變異點：已存在的略過。
4. 完成後會顯示各類型合併筆數。

## 常見問題

**Q：換手機怎麼搬資料？**
在舊手機「匯出分享」存到雲端硬碟，新手機安裝 app 後「匯入合併」即可。
（資料集需另行重新下載。）

**Q：刪除資料集會弄丟我的紀錄嗎？**
不會。追蹤中的點位與其紀錄、照片在刪除資料集時都會保留。

**Q：定位拿不到位置？**
確認系統已授予 app 定位權限、GPS 已開啟；在室內收訊不佳時可先到
開闊處再按「我的位置」。
